## User Manual 

### Install PyTest 
    $ pip install pytest==2.9.1

### Run 
- To run all test cases:  
    `$ pytest Test_Cases.py`
- To run select cases of a specific marker (i.e. category):  
    `$ pytest -m Harir Test_Cases.py`  
    (NOTE: See `pytest.ini` for all categories)
